Increase Blog Revenue via AI Product Selection Strategy

== Description ==

Moredeal plugin will help monetize your WordPress content and greatly increase revenue in a short time. This is a perfect tool for blog content creators and help them select recommended products more conveniently, it can greatly improve product selection efficiency for both blog creators and affiliates.

= What is this plugin for? =

* Diversified Database - Collect the best sellers on Amazon and gather hot items on deal websites, SNS platforms & professional reviews.
* Sufficient Information - Display product image, price, sales count, rating, review and rank.
* Auto Update - The prices and stocks will be updated automatically.
* Products Hidden - Hide products or prices in Moredeal settings when they are out of stock.
* Affiliate Links - Select the searching products and affiliate links will be generated.
* Display Types - Select different styles to display products and keep outstanding.

Official website: [mdc.ai](https://mdc.ai/ "mdc.ai")
Documentation: [docs.mdc.ai](https://docs.mdc.ai/ "docs.mdc.ai")

= How it works? =

* Activate your modules
* Enter the product keywords
* Select product strategies
* Add offers
* Start to earn


= Plugin features =

* Multiple data sources like deal websites, SNS platforms and professional reviews.
* AI product selection strategies to find items more efficiently and accurately.
* Recommend products by different product selection strategies.
* Search products via keywords, Amazon product link or ASIN to find the best items.
* Filter products through rating, reviews, prices, sales count, sales etc.
* Automatic update prices and stocks.
* Select the searching products and affiliate links will be generated.
* Easy interface: add all contents from post edit page.
* Select different styles to display products and keep outstanding.
* Works with any theme.
* Works with Wordpress shortcodes.

== Installation ==

**Requirements**

* PHP version 5.4 or higher.
* Wordpress 4.6.1 or higher.

This section describes how to install the plugin and get it working.

1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure plugin settings
1. You can find manual how to configure each module - [Moredeal User Guide](https://docs.mdc.ai/)
